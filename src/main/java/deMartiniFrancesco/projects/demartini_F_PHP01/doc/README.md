## Consegna
Sull’ app di gestione alunni:
-Creare un form di ricerca per la classe, la pagina di risposta deve visualizzare la classe in ordine alfabetico e
il numero totale degli alunni.
-Creare una pagina che permetta di modificare la classe di un alunno.
-Creare una pagina che permetta, dopo un controllo input, di aggiungere un nuovo alunno.
